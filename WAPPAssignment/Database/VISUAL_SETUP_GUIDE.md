## ? SUCCESS CRITERIA

You've successfully set up the database when ALL are true:

- ? CosmoQuestDB.mdf exists in App_Data folder
- ? Database connection shows in Server Explorer
- ? 10 tables visible when expanded
- ? Users table has 1 record (admin)
- ? Planets table has 8 records
- ? StarsGalaxies table has 5 records
- ? SpaceMissions table has 5 records
- ? Quizzes table has 1 record
- ? QuizQuestions table has 5 records
- ? Application runs without errors
- ? Can login as admin/Admin@123
- ? Admin Dashboard shows statistics
- ? Solar System page shows 8 planets
- ? No database connection errors in browser

To start over:

```

   ? Right-click CosmoQuestDB.mdf
   ? Click "Delete"

   ? CosmoQuestDB.mdf
   ? CosmoQuestDB_log.ldf
5. Start again from Step 2
```
